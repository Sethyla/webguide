HTML & CSS Guide

Keep in mind that it always has to be "<>" for it to count as a "function" in HTML. When closing a line, like in C# where we use ";" in programming, it's similar to the opening block but with a "/" before the word. An example is shown below.

It is important to always include DOCTYPE at the top of your HTML file. This tells the file that the language being used is HTML.

Another important thing is that after writing DOCTYPE, you open with <html>. This tells the browser that all code inside <html> is written in HTML.

You can also add lang="sv" (or "en") inside <html>, which tells the browser what language the website is in. "sv" stands for Swedish, "en" for English. There are abbreviations for all languages, but these are the most important ones.

<!DOCTYPE html>
<html lang="sv">
Input Code:





</html>

There are 4 different "text" types: H1, H2, H3, and p. H1-H3 are headers whose sizes cannot be changed. The "p" tag stands for Paragraph, used for writing regular text, like this box you are reading now. The difference between H1-H3 is the size of the header. A header can also be called a title or subheading.

H1


H2


H3


P



<!DOCTYPE html>
<html>
<h1> Hello </h1>
<h2> Hello </h2>
<h3> Hello </h3>
<p> Hello </p>
</html>

There are 3 types of elements that come after <html>: Body, Head, and Style. The difference is that Body is what is visible, Head is what happens in the background, still important but not seen the same way as Body. For example, <title> sets the tab title when the website is open. Style is a version of CSS which I will explain later. The Body contains all headers, paragraphs, and anything else you want visible on the page.



<!DOCTYPE html>
<html>
<head>
<title>
tab title
</title>
</head>
<body>
<h1>
Hello
</h1>
</body>
</html>

As I mentioned before, there are 3 types of CSS: inline, internal, external. Inline is applied directly in the element, internal is written inside <style> tags, and external comes from a separate file, such as style.css.

The example next to this shows how to make H1 text black. Remember, colors must be written in English if you want to use the built-in names.

Another example sets the body background color to white.

External CSS works like internal CSS, but you have to load it from a separate file, for example style.css. The example also shows how to import CSS from another file.



<!DOCTYPE html>
<html>
<link rel="stylesheet" href="style.css">
<style>
body {
background-color: white;
}
</style>
<head>
<title>
tab title
</title>
</head>
<body>
<h1 style="color:black;">
Hello
</h1>
</body>
</html>

Adding images is simple. It's important to create a folder to keep them organized. In this example, the folder is called "images" and contains our images. Give each image a short and simple name to make it easier.

Open an image with <img> and set the src attribute to the file path, e.g. "images/Bat.png". Include the file extension (jpg, png, etc.). The alt attribute provides text if the image cannot load.



<!DOCTYPE html>
<html>
<body>
<img src="images/Bat.png" alt="Bat">
</body>
</html>
