HTML & CSS Guide

Keep in mind that it always has to be "<>" for it to count as a "tag" in HTML.

When closing a line, like in C# where we use ";" in programming, it's similar to the opening block but with a "/" before the word. An example is shown below.

A tag is a function. Just like in C# you have int, double, string, etc., in HTML you have tags like p, h1, h2, h3, img, and more.

It is important to always include DOCTYPE at the top of your HTML file. This tells the file that the language being used is HTML.

Another important thing is that after writing DOCTYPE, you open with <html>. This tells the program that all code inside <html> is written in HTML.

You can also add lang="sv" or lang="en" inside <html>, which tells the browser what language the website is in. "sv" stands for Swedish, "en" stands for English. There are abbreviations for all languages, but these are the most important. 

<!DOCTYPE html>
<html lang="sv">
Input Code:






</html>

There are many different "text" types, and among these are: H1, H2, H3, and p. H1-H3 are headers of diffrent sizes. The "p" tag stands for Paragraph, used to write blocks of text, like this box you are reading now. The difference between H1-H3 is the size of the header. A header can also be called a title or subheading.

H1

H2

H3

P

<!DOCTYPE html>
<html>
    <h1> Hi </h1>
    <h2> Hi </h2>
    <h3> Hi </h3>
    <p> Hi </p>
</html>

There are 2 types of elements go inside <html>: Body and Head. The difference is that while Body is what is visible on the main page, Head is what happens in the background, still important but not seen the same way as Body. For example, <title> sets the tab title when the website is open. The Body contains all headers, paragraphs, and anything else you want visible on the page. 

<!DOCTYPE html>
<html>
    <head>
        <title>Tab Title</title>
    </head>
    <body>
        <h1>Hi</h1>
    </body>
</html>

As mentioned before, there are 3 types of CSS: inline, internal, and external. Inline is applied directly in the element, internal is written inside <style> tags within the head, and external comes from a separate file, like style.css.

The example next to this shows how to make H1 text black. Remember, colors must be written in English if you want to use the built-in names.

Another example sets the body background color to white.

External CSS works like internal CSS, but you have to load it from a separate file, for example style.css. The example also shows how to import CSS from another file. This may be considered advanced at the time of writing. 



<!DOCTYPE html>
<html>
    <head>
        <title>Tab Title</title>
        <link rel="stylesheet" href="style.css">
        <style>
            body {
                background-color: white;
            }
        </style>
    </head>
    <body>
        <h1 style="color:black;"Hi</h1>
    </body>
</html>

Adding images is simple. It's important to create a folder to keep them organized. In this example, the folder is called "images" and contains our images. Give each image a short and simple name to make it easier.

Open an image with <img> and set the src attribute to the file path, e.g. "images/bat.png". Include the file extension (jpg, png, etc.). The alt attribute provides text if the image cannot load. 

<!DOCTYPE html>
<html>
    <body>
        <img src="images/bat.png" alt="bat">
    </body>
</html>
